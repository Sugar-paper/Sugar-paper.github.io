安卓机搭建Linux web服务器

发表于2019-10-24|更新于2020-03-07
|阅读量:

android安装linux系统

  1. 下载并安装 Linux deploy ，对其设置后台运行权限，并锁定程序。国改系统请依照系统内部的管理程序锁定后台，例如MIUI中将其移出神隐模式.

  2. 给安卓手机获取root权限。（如果你这一步都不会的话建议不要继续下去了，去论坛逛逛）

    这里给出两个方法，

    • 通过刷TWRP后刷入第三方系统然后刷入面具Magisk
    • 通过官方发布的开发版或内测版的系统获取
    • 使用刷机软件获取（通常会失败，不推荐）

  3. 安装busybox，各大应用商店均可搜索下载。

    到这里准备工作就基本完成了。

安装Linux镜像 地址

  1. 打开deploy点击右下角的设置按钮进行配置文件的设置。
容器类型 chroot 属性
容器类型 chroot 更稳定
发行版GNU/Linux Ubuntu 拥有arm64版
架构 Arm64 安卓手机arm/aarch架构
发行版GNU/Linux版本 bionic 18.04
源地址 http://mirrors.aliyun.com/ubuntu-ports/ 快，可用，经测试其他镜像均有问题
安装类型 镜像文件 保证系统的独立性与完整性，且不额外占用空间，安卓下可直接读取
安装路径 ${EXTERNAL_STORAGE}/dir/name.img 一定要为img结尾
镜像大小 32768 32GB，够大够用，且手机可用空间只会占用实际大小
文件系统 Ext4 保持与TWRP中设置的文件系统一致即可
用户名 自定 普通用户登入名
用户密码 自定 xxx
特权用户 自定 root用户登录名
本地化 zh_CN.UTF-8 字符集
初始化 启用
挂载 启用 读取安卓内存
挂载列表 /sdcard/ 内存
SSH 启用 远程访问服务器
VNC图形界面 可选 图形化访问
桌面环境 LXDE
(使用原生gnome桌面)见跳转		 openbox
音频 可选 声音输出
  1. 点击主界面左上角菜单→设置，勾选屏幕常亮、锁定WIFI、CPU唤醒。path变量，填入busybox地址，例如我的/system/xbin/。
  2. 选择好配置后回到主界面点击右上角的菜单，选择安装接着就是漫长的等待。直到输出<<deploy代表安装完成。
  3. 点击配置（只需一次）。如需进行更新环境需点此项进行更新。
  4. 点击运行。
  5. 附加：centos可用的源http://mirrors.ustc.edu.cn/centos-altarch/

远程连接Linux/Ubuntu

打开终端输入ssh 用户名@IP地址。

输入sudo apt-get install ttf-wqy-zenhei安装中文字体包。

输入vncpasswd更改密码后可通过vncviwer进行可视化连接。

安装开发包

在特权用户下（或sudo）安装开发组件：

  • apt-get install firefox

    火狐浏览器，如果安装Ubuntu-desktop请略过（修改为中文）

  • apt-get install build-essential

    build-essential 软件包，它提供了编译程序必须的软件包。

  • apt-get install libtool

    libtool

  • apt-get install wget 命令安装 wget

  • ```bash
    apt install zlib1g-dev gcc make git autoconf autogen automake pkg-config uuid-dev
    git clone https://github.com/netdata/netdata.git –depth=100

    cd netdata

    ./netdata-installer.sh

    #x-window-manager &
    gnome-panel &

    export XKL_XMODMAP_DISABLE=1

    然后在终端执行
